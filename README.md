# AI Chatbot

A simple AI chatbot built using React, Node.js, Express, MongoDB, and the Google Gemini API. This project allows users to interact with an AI assistant through a clean and responsive web interface.

## Features

* AI-powered conversations using the Google Gemini API
* Clean and responsive user interface
* React frontend with Vite
* Node.js and Express backend
* MongoDB integration

## Tech Stack

**Frontend**

* React.js
* Vite
* Tailwind CSS

**Backend**

* Node.js
* Express.js

**Database**

* MongoDB

**AI**

* Google Gemini API

## Installation

### Clone the repository

```bash
git clone https://github.com/shashank27589-blip/Chatbot.git
cd Chatbot
```

### Install dependencies

Backend:

```bash
cd backend
npm install
```

Frontend:

```bash
cd ../frontend
npm install
```

### Configure environment variables

Create a `.env` file inside the `backend` folder.

Example:

```env
MONGODB_URI=your_mongodb_connection_string
GEMINI_API_KEY=your_gemini_api_key
PORT=5000
```

### Run the project

Start the backend:

```bash
cd backend
npm start
```

Open a new terminal and start the frontend:

```bash
cd frontend
npm run dev
```
